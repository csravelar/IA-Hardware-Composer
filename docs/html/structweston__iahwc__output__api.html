void(* weston_iahwc_output_api::set_gbm_format) (struct weston_output *output, const char *gbm_format)

The pixel format to be used by the output. Valid values are:

  • NULL - The format set at backend creation time will be used;
  • "xrgb8888";
  • "rgb565"
  • "xrgb2101010"
