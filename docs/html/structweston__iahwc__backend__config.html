◆ gbm_format

char* weston_iahwc_backend_config::gbm_format

The pixel format of the framebuffer to be used.

Valid values are:

  • NULL - The default format ("xrgb8888") will be used;
  • "xrgb8888";
  • "rgb565"
  • "xrgb2101010" The backend will take ownership of the format pointer and will free it on backend destruction.

uint32_t weston_iahwc_backend_config::pageflip_timeout

Maximum duration for a pageflip event to arrive, after which the compositor will consider the IAHWC driver crashed and will try to exit cleanly.

It is exprimed in milliseconds, 0 means disabled.
